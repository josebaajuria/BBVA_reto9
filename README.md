# Reto\_09\_Azul\_Claro

Cosas a tener en cuenta:

* Los scripts se deben de ejecutar en orden.
* Se necesitan los csv otorgados por los pm y Lanit para poder resolver el reto.

## Datos

En estas carpetas estan o se tienen que ubicar todos los datos utilizados.

### Originales

Aquí se tienen que incluir todos los datos que se han otorgado al principio del reto.
<br><br>
## \_01\_Ingesta\_Limpieza.py

En este script, se cargan y se limpian todos los datos que se van a utilizar en el reto.

## \_02\_Red\_Neuronal\_Principal.py
Este script ejectura la red neuronal y devuelve dos csv, el de datos sintéticos y el de resultados

## Red\_Neuronal\_Con\_Graficos.ipynb
En este script se hacen los gráficos de la red neuronal (definida en el anterior script) y se muestran diferentes resultados.

## MonteCarlo\_y\_Formula\_cerrada\_25\_26.ipynb
Script dado por BBVA, contiene la fórmula cerrada y el función de Monte Carlo dado por BBVA.

## Calculo\_Formula\_Cerrada.ipynb
Cálcula la prima de la fórmula cerrada y explica la fórmula cerrada.

## Simulacion\_Monte\_Carlo\_AZUL\_CLARO.ipynb
Monte Carlo creado por nosotros, incluye la prima sacada.

## indicaciones\_entorno\_virtual\_conda.txt
Indicaciones para crear el entorno virtual

## entorno\_AZUL\_CLARO\_R9.yml
Entorno necesario para ejecutar todo el proyecto.
